# 尖塔储君卡图娘化 (RegentFemPortraits)

将《杀戮尖塔2》储君角色的卡牌立绘替换为精美的二次元风格卡图。

## 功能特性

### 卡图替换
- **40张普通卡图替换**：覆盖储君职业的大部分卡牌
- **1张先古样式卡图**：君王之剑(SovereignBlade)可选择显示为先古之民风格
- **线性各向异性过滤**：应用最佳的纹理过滤模式，减少锯齿，提升画质

### 可配置项
当安装 [BaseLib 3.1.0+](https://github.com/Alchyr/BasLib/releases) 时，可在游戏设置中配置：
- **SovereignBladeAncientStyle**：控制君王之剑是否显示为先古之民样式
  - 开启：显示为先古样式（带特殊边框和背景）
  - 关闭：显示为普通替换卡图

### 依赖说明
- **软依赖 BaseLib**：本模组不依赖 BaseLib 即可正常运行所有卡图替换功能
- 安装 BaseLib 后会额外显示先古样式配置选项

## 安装说明

1. 下载最新版本的模组文件
2. 将模组文件夹放入游戏的 `mods` 目录
3. 启动游戏，模组将自动加载
